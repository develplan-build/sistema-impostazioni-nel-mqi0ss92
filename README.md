# CasaPro - Real Estate Management SaaS

CasaPro is a professional B2B SaaS for real estate agencies to manage properties, clients, appointments, contracts, and reports.

## What you received
The online site you are viewing is a **demo** running entirely in the browser with local state. This repository contains the **complete, production-ready source code**. To make it real, you need to connect your own services (Database, Payments, Emails) by following this guide.

## Requirements
- Node.js 18+
- A Supabase account (Database & Auth)
- A Stripe account (Payments)
- A SendGrid account (Emails)
